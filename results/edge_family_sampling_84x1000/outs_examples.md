# Expanded Dead-Card Outs Examples

These examples are built from the published `84 x 1000` run. The game tree is still standard hero-vs-dealer UTH; the `10` exposed cards are treated as known dead cards.

An **out** here means an exposed/dead card whose rank matches either hero rank. So for `KQo`, any dead `K` or dead `Q` counts as one out.
